## Keyboard Navigation Flow

```
Tab Order:
1. [A] Alice judgment (collapsed)
   ↓ Press Enter or Space to expand
2. [B] Bob judgment (collapsed)
   ↓ Press Enter or Space to expand
3. [C] Charlie judgment (collapsed)
   ↓ Press Enter or Space to expand

Focus Ring:
┌─────────────────────────────────────────────────────────┐
│ [A] Alice        🌟 대성공                          ▼  │ ← Blue ring
└─────────────────────────────────────────────────────────┘
```

## Real-World Usage Example

```tsx
// In JudgmentModal.tsx
const completedJudgments = [
  {
    action_id: 1,
    character_id: 101,
    character_name: "Alice",
    action_text: "거대한 바위를 들어올립니다",
    ability_score: "str",
    modifier: 5,
    difficulty: 20,
    difficulty_reasoning: "매우 무거운 바위입니다",
    status: "complete",
    order: 0,
    dice_result: 20,
    final_value: 25,
    outcome: "critical_success",
    outcome_reasoning: "완벽한 힘으로 바위를 들어올렸습니다!"
  },
  {
    action_id: 2,
    character_id: 102,
    character_name: "Bob",
    action_text: "적을 설득하려고 합니다",
    ability_score: "cha",
    modifier: 1,
    difficulty: 18,
    difficulty_reasoning: "적이 매우 화가 나 있습니다",
    status: "complete",
    order: 1,
    dice_result: 1,
    final_value: 2,
    outcome: "critical_failure",
    outcome_reasoning: "말실수로 상황이 더 악화되었습니다"
  }
];

<CompletedJudgmentsList judgments={completedJudgments} />
```

This will render a beautiful, interactive list of completed judgments with full keyboard and screen reader support!
